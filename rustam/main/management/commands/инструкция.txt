Запуск
 - bot.exe

В конфиге
 - token - токен бота.
 - merchant_id - id мерчанта в kaspi, чтобы игнорировать его, если он с самой низкой ценой на рынке. Указывать через , если их несколько.
 - telegram_id - id админов, кто пользуется ботом, через ,

Как получить токен бота
1. Вводим в поиске в телеграме "botfather"
2. Начинаем чат с ботом
3. Вводим /newbot
4. Следуем инструкции от бота:
  - Вводим имя бота, которое будет выводится для всех
  - Вводим username бота, важно чтобы он оканчивался на "bot" к примеру test_bot, если username занят, то попробуйте другой.
  - В ответ получим сообщение об успешном создании бота, где после строки Use this token to access the HTTP API: лежит наш токен.
5. Копируем токен, вставляем в таблицу и можно начинать чат с нашим ботом, по ссылке которая была при создании бота
